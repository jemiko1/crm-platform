CRM28 Phone — UI direction mockups

Four distinct visual directions for the softphone. Each one shows three key states: Idle (dialpad visible), In-call (active call with caller info), and On break (fullscreen modal). Pick whichever feel you want me to run with and I'll do the full polish pass on the real app. Mix & match is fine too — e.g. "Focus layout but Glass colors".

1. Focus

Apple-style minimal. White. One accent color. Generous whitespace.

Vibe: calm, precise, no visual noise. Think iPhone dialer.
Color: white background, single blue accent (#2563eb), text is near-black.
Who it suits: operators who prefer "get out of my way" UI.

What each direction costs:

  • Focus — smallest code change, easiest to maintain. Minimal risk.
  • Glass — backdrop-filter is Electron/Chromium-only (that's fine, we're already Electron). Bigger CSS surface.
  • Command — biggest opinion-change. Would also need a new "stats today" backend endpoint for the header row (quick call-center metrics).
  • Warm — pure CSS. Mood shift is the biggest ask; some users may feel it's "too casual" for a business tool.
